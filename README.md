# Movie App

## Development


To install and work on `https://github.com/halilatilla/movie-app`

```bash
git clone https://github.com/halilatilla/movie-app
cd movie-app
npm install or yarn install
```

<br>

```bash
yarn dev
```

`http://localhost:3000/`
